# Security Policy

## Public Information Only

This repository must not contain private account information, API keys, tokens, payment details, server credentials, affiliate dashboards, or personal identifiers.

If you accidentally publish sensitive information:

1. Remove it immediately.
2. Rotate the exposed credential or token.
3. Open a private report with the maintainer if possible.

## Reporting

Please open an issue only for public, non-sensitive documentation problems. Do not paste secrets, account screenshots, invoices, support tickets, or payment information into an issue.
