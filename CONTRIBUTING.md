# Contributing

Thanks for helping improve this Cloudways pricing reference.

## Good Contributions

- Correct outdated wording with official Cloudways sources.
- Improve explanations of Flexible, Autonomous, billing, payment, or WordPress hosting choices.
- Add neutral decision checklists.
- Fix broken links, typos, or confusing phrasing.

## Not Accepted

- Affiliate-only pages or promotional link drops.
- Claims that this is an official Cloudways repository.
- Unsourced price tables.
- Private screenshots, account information, API keys, payment details, or support tickets.
- One-sided claims that a single plan is best for every user.

## Source Rules

Prefer official Cloudways pages or documentation. If you use another source, explain why it is reliable and what part it supports.

## Pull Request Checklist

- [ ] The change is useful for pricing, billing, payment, or WordPress hosting decisions.
- [ ] Official sources are linked when facts may change.
- [ ] No private information or credentials are included.
- [ ] Links are relevant and not spammy.
- [ ] The wording does not imply official Cloudways ownership.
