## Project Summary

A practical reference for understanding Cloudways pricing, billing, and WordPress hosting choices. This repository is for people who want to compare Cloudways plans without turning the decision into a pure coupon hunt. It summarizes how the pricing model works, what to check before choosing a plan, and where to read more detailed setup guidance.

## Current Notes

Last checked: 2026-06-05.

The official Cloudways pricing page presents two major routes:

- **Flexible**: configurable managed hosting where you choose cloud provider, server size, and region.
- **Autonomous**: fully managed WordPress hosting with autoscaling for higher-traffic sites.

The official billing guide explains that Cloudways uses a pay-as-you-go model. Flexible is charged hourly for running servers and summarized in a monthly invoice. Autonomous is billed hourly per active application, including autoscaling usage.

This repository intentionally does not mirror every live price row. Use it as a decision guide, then confirm live price, region, promotion, and invoice details in the official Cloudways interface.
