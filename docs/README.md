# Cloudways Pricing, Billing and WordPress Hosting Guide

This GitHub Pages site is a practical Cloudways reference for people comparing pricing, billing, payment, checkout notes and WordPress hosting routes. It is designed to help you understand the decision before opening the official checkout.

Final prices, regions, providers, promotions, invoices and add-ons can change. Always confirm the final amount inside the Cloudways official pricing page, dashboard or checkout.

## Quick Decision Path

1. Decide whether your project is a new WordPress site, WooCommerce store, agency site, content site or higher-traffic application.
2. Compare Flexible and Autonomous separately.
3. Check provider, server size and region before looking at coupons.
4. Review add-ons such as backups, email, DNS, CDN and security tools.
5. Confirm payment method and invoice behavior before launching production workloads.
6. Verify final price and available promotions in the official Cloudways checkout.
