## 中文用户推荐判断顺序

1. 先判断网站类型：展示站、内容站、外贸站、WooCommerce、客户项目还是高流量站。
2. 新站优先从 Flexible 路线理解，不要一开始就只看高配或自动扩展。
3. 机房按目标用户所在地选，不按站长所在地选。
4. 结账前核对云厂商、服务器大小、地区、备份、邮件、DNS、CDN、安全等附加项。
5. 确认付款方式、账单周期、优惠是否在官方结账页实际生效。
