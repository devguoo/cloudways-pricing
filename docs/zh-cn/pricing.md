# Cloudways 价格和套餐说明

最后核对：2026-06-05。

Cloudways 价格不能只看一个“最低月费”。真正购买前要同时看产品路线、云厂商、服务器配置、机房、运行时间、备份和附加服务。

如果你想先看完整中文入口，可以回到：[Cloudways 中文价格、付款与 WordPress 主机入口](./)。

## 1. 两条主要路线

### Flexible：可控配置路线

Flexible 更适合大多数中文用户先看：

- 新 WordPress 站。
- 外贸展示站。
- 企业官网。
- 内容站。
- 代理建站项目。
- 预算需要可控的项目。

它的特点是你可以选择底层云厂商、服务器大小和机房，后续按真实访问、后台速度和订单情况升级。

### Autonomous：自动扩展路线

Autonomous 更适合已经有流量压力或动态业务的 WordPress 站点：

- WooCommerce 店铺。
- LMS/会员站。
- 活动流量明显的站点。
- 不想手动判断扩容的团队。

普通新站不建议因为它看起来更高级就直接选择。

## 2. 影响账单的因素

购买前至少要看：

- 选择的是 Flexible 还是 Autonomous。
- 底层云厂商和服务器大小。
- 机房位置。
- 服务器或应用实际运行时间。
- 备份、邮件、DNS、CDN、安全等附加服务。
- 带宽、存储、自动扩展等额外使用。
- 税费、账户余额、优惠或活动。

## 3. 推荐判断顺序

1. 先确认自己是否能完成付款。
2. 再判断网站是展示站、内容站、外贸站、WooCommerce 还是客户项目。
3. 新站优先从 Flexible 可控配置路线开始看。
4. 机房按目标用户所在地选，不按站长所在地选。
5. 有真实访问、订单、后台卡顿或资源压力后，再升级。
